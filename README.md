# BreakingBadMVVM

A simple project build in Swift using Auto Layout Visual Format Language (VFL) under
ModelView View Model (MVVM) architecture.

## Goals
VFL

MVVM

No Libs

Base URL stored xcconfig

Unit test for MVVM

Unit test for ApiService
